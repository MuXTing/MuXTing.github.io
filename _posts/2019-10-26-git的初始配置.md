记住以下这三个命令:
> 1. git config --global user.name "examplename"
  2. git config --global user.email exampleEmail@outlook.com

上面是配置好个人信息
> 3. ssh-keygen -t rsa -C "exampleEmail@outlook.com"

生成公钥，填写到Github上。
